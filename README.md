This project is a premium frontend-only weather dashboard built with React and Axios. It is a static client-side application with no backend or database. The app consumes external APIs to provide live weather information and visual city content in a polished modern interface.

APIs used:
- OpenWeather API: used to fetch current weather, 24-hour / forecast data, humidity, pressure, wind speed, visibility, sunrise, and sunset information.
- Pixabay API: used to fetch city-related images displayed in the weather cards and detail views.

Main functionalities:
- Search and add cities manually
- Display current weather for selected cities
- Show city cards with live weather and image background
- View detailed weather page for each city
- Display 24-hour forecast and weather trends
- Show weekly forecast section
- Display dashboard highlights such as humidity, wind speed, feels like, pressure, and visibility
- Show chance of rain widget with visual bars
- Smart weather alerts for rain, heat, and wind conditions
- Notifications page for weather alert monitoring
- Profile page with personalized weather space
- AI weather assistant panel that gives dynamic weather insights and recommendations
- Responsive premium UI inspired by modern weather applications
